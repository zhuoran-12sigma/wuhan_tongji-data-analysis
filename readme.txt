1. 请将所有病人的金标准存入一个文件夹， 文件名形式 "病人序号.json", 如“111.json"
2. 请将所有CAD检测结果按序列存入一个文件夹，文件夹下有12个序列的子文件夹，每一个子文件夹下存入同一序列所有病人的检测结果， 文件名形式“病人序号_序列名.json”, 如”111_std60.json“
3. 请保证计算机已安装python 2或 python 3 及 numpy package
4. 打开terminal， 输入 
	python 程序路径/AutoAnalysis.py
   注意，若是window系统请将”/" 改为“\”
	
   按照出现的提示命令输入文件路径，每个路径以“/”(Linux 或 MacOS系统) 或“\\” (windows系统)结束
   敲回车键即可运行程序
